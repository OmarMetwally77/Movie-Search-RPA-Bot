# Movie Search Automation Bot

An RPA automation project built using UiPath that reads movie names from Excel, searches for them on a website, and writes the search results automatically.

## Features
- Excel Automation
- Browser Automation
- Automated Movie Search
- Found / Not Found Detection
- Execution Summary Report
- Email Notification Automation

## Technologies Used
- UiPath
- Excel Activities
- Browser Automation
- DataTables
- Conditional Logic

## Workflow
1. Read movie names from Excel
2. Search each movie on website
3. Detect result
4. Write result to Excel
5. Generate execution summary
6. Send email report automatically

## Project Summary
The bot automates the process of validating movie names from an Excel sheet through a website search process and generates a final execution report including success statistics.
